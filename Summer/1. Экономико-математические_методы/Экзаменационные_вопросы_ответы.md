### Вопрос 1. Подробная классификация ЭММ в АПК по признакам: цель моделирования, учет фактора времени, вид математического аппарата

| Тип модели | Суть | Пример в АПК |
|------------|------|-------------|
| Оптимизационные | Поиск наилучшего решения (max прибыли, min затрат) | Оптимальное сочетание отраслей с/х предприятия, оптимальный рацион кормления |
| Балансовые | Отражение равенства между ресурсами и их использованием | Межотраслевой баланс АПК региона, баланс кормов |

| Описательные (дескриптивные) | Описание и объяснение наблюдаемых явлений | Производственные функции, корреляционно-регрессионные модели урожайности |
| Прогнозные | Предсказание будущих значений показателей | Прогноз урожайности на основе трендов, прогноз цен на зерно |

**По учету фактора времени:**

| Тип модели | Суть | Пример в АПК |
|------------|------|-------------|
| Статические | Описывают экономику в фиксированный момент, без учёта динамики | Оптимизация посевных площадей на один год, статический МОБ |
| Динамические | Учитывают изменения во времени, включают несколько периодов | Многолетний план развития стада, динамический МОБ с учётом инвестиций |

**По виду математического аппарата:**

| Тип модели | Суть | Пример |
|------------|------|--------|

| Теории игр | Принятие решений в условиях конфликта/неопределённости | Выбор стратегии посева при неизвестной погоде |
| Сетевые модели | Графы и сети | Планирование и контроль строительства с/х объектов |
| Корреляционно-регрессионные | Статистические зависимости | Зависимость урожайности от количества удобрений и осадков |

### Вопрос 2. Системный подход к построению ЭММ в АПК. Особенности аграрной экономики на каждом этапе

**Системный подход** — рассмотрение АПК как целостной системы взаимосвязанных элементов (земля, труд, техника, биологические объекты, рынки сбыта).

**Этапы построения и специфика АПК:**

1. **Постановка задачи и определение цели.**
   - Специфика АПК: необходимо учитывать многоотраслевой характер (растениеводство, животноводство, переработка), сезонность, зависимость от природных факторов.

2. **Качественный анализ объекта** — изучение структуры, связей, факторов.
   - Специфика: биологические процессы (рост растений, развитие животных), зависимость от климата и почвы, длительный производственный цикл.

3. **Формализация задачи** — перевод экономической задачи на язык математики (переменные, ограничения, целевая функция).
   - Специфика: севооборотные ограничения, агротехнические требования, нормы кормления, сезонность трудозатрат.

4. **Анализ количественных зависимостей** — определение числовых параметров модели (нормативы, коэффициенты).
   - Специфика: технологические карты возделывания культур, нормы расхода кормов, зоотехнические нормативы, данные по урожайности.

5. **Решение модели** — применение математических методов (симплексный метод, метод потенциалов и др.).
   - Специфика: большая размерность задач АПК, необходимость целочисленных решений (поголовье, число тракторов).

6. **Анализ результатов и внедрение** — экономическая интерпретация, проверка адекватности, корректировка.
   - Специфика: результаты должны быть агрономически и зоотехнически реализуемыми; необходимо учитывать рыночную конъюнктуру и государственное регулирование.

### Вопрос 3. Преимущества и недостатки статических и динамических моделей в АПК

**Статические модели:**

| Преимущества | Недостатки |
|-------------|-----------|
| Простота построения и решения | Не учитывают изменения во времени |
| Меньше данных для параметризации | Не отражают накопление/износ ресурсов |
| Быстрый результат | Не видна динамика развития |
| Пригодны для краткосрочного планирования | Не учитывают инвестиционные решения |

**Целесообразны:** оперативное и годовое планирование (оптимизация посевных площадей на год, распределение кормов на стойловый период, расчёт рационов).

**Динамические модели:**

| Преимущества | Недостатки |
|-------------|-----------|
| Отражают развитие процесса во времени | Более сложны в построении и решении |
| Учитывают инвестиции, воспроизводство стада | Требуют больше данных (прогнозные цены, тренды) |
| Позволяют оптимизировать стратегию на перспективу | Накопление ошибок на длинном горизонте |
| Учитывают сезонность, севообороты | Высокая вычислительная сложность |

**Целесообразны:** стратегическое планирование (развитие стада на 5–10 лет, инвестиционные проекты, перспективное размещение предприятий, динамический МОБ).

### Вопрос 4. Этап формализации экономической задачи. Типичные ошибки

**Формализация** — перевод содержательного описания задачи в математическую форму: определение переменных, запись целевой функции и системы ограничений.

**Содержание этапа:**
1. Выбор переменных решения (что ищем — объёмы производства, площади, поголовье)
2. Формулировка целевой функции (критерий оптимальности — прибыль, затраты, валовая продукция)
3. Запись системы ограничений (ресурсные, технологические, рыночные, нормативные)
4. Определение области допустимых значений переменных

**Типичные ошибки:**

| Ошибка | Как избежать |
|--------|-------------|
| Неполный учёт ограничений (забыли севооборот, сезонность труда) | Тщательный качественный анализ объекта до формализации |
| Линеаризация нелинейных зависимостей без обоснования | Проверять характер зависимости, использовать кусочно-линейные аппроксимации |
| Смешение размерностей (руб., тонны, га в одном ограничении) | Контроль размерностей на каждом шаге |
| Избыточная детализация (слишком много переменных) | Принцип существенности — включать только значимые факторы |
| Недостаточная детализация (агрегирование разнородных культур) | Разделять культуры/отрасли, если нормативы существенно различаются |
| Неверное определение целевой функции (max выручки вместо max прибыли) | Чёткая экономическая постановка цели |

### Вопрос 5. Адекватность ЭММ. Методы и критерии проверки в АПК

**Адекватность модели** — степень соответствия модели реальному экономическому объекту, способность модели воспроизводить существенные свойства оригинала.

**Методы проверки адекватности:**

1. **Ретроспективная проверка** — решение модели на данных прошлых лет и сравнение с фактическими результатами.

2. **Экспертная оценка** — специалисты-аграрники оценивают реалистичность оптимального плана (например, рекомендованная структура посевных площадей должна быть агрономически обоснованной).

3. **Анализ двойственных оценок** — теневые цены ресурсов должны иметь экономический смысл (нулевая оценка для избыточного ресурса, положительная — для дефицитного).

4. **Параметрический анализ (анализ чувствительности)** — проверка устойчивости оптимального решения при изменении параметров модели в пределах возможных отклонений.

5. **Статистические критерии** — для регрессионных моделей: коэффициент детерминации R², критерий Фишера, t-критерий Стьюдента.

6. **Сравнение с аналогами** — сопоставление результатов с показателями передовых хозяйств.

**Причины неадекватности:** неполный учёт факторов, устаревшие нормативы, игнорирование нелинейности, неверная спецификация модели.

### Вопрос 6. Иерархическая система моделей в АПК. Согласование уровней

**Уровни иерархии:**

| Уровень | Объект моделирования | Типичные задачи |
|---------|---------------------|----------------|
| Федеральный | АПК страны | Продовольственная безопасность, размещение производства по регионам, экспорт/импорт |
| Региональный | АПК региона | Специализация районов, размещение перерабатывающих предприятий, межрайонные перевозки |
| Предприятие | Отдельное хозяйство | Структура посевных площадей, оптимальный рацион, производственная программа |

**Принципы построения:**
- Декомпозиция — разделение общей задачи на подзадачи по уровням
- Координация — обмен информацией между уровнями
- Итеративность — последовательное уточнение решений

**Согласование уровней:**

1. **Сверху вниз (директивный):** федеральный уровень задаёт контрольные цифры (объёмы госзакупок, квоты), регион распределяет по районам, район — по хозяйствам.

2. **Снизу вверх (агрегирование):** хозяйства формируют оптимальные планы, они агрегируются на уровне района, региона, страны.

3. **Итеративное согласование:** циклический обмен — верхний уровень даёт ориентиры, нижний рассчитывает возможности, результат передаётся наверх для корректировки. Повторяется до сходимости (метод Данцига–Вулфа, метод декомпозиции).

4. **Через двойственные оценки:** оценки ресурсов с верхнего уровня передаются как цены на нижний уровень, что обеспечивает экономическую координацию.

### Вопрос 7. Оптимизационные, балансовые и имитационные модели в АПК

**Оптимизационные модели:**
- Находят наилучшее решение при заданных ограничениях
- Математический аппарат: линейное, нелинейное, целочисленное, динамическое программирование
- Примеры: оптимизация структуры посевных площадей (ЛП), оптимальный рацион кормления (ЛП), оптимальное размещение перерабатывающих предприятий (целочисленное ЛП), транспортная задача перевозки зерна

**Балансовые модели:**
- Отражают равновесие между производством и потреблением ресурсов
- Математический аппарат: система линейных уравнений, матричная алгебра
- Примеры: межотраслевой баланс (МОБ) АПК региона, баланс кормов хозяйства, баланс трудовых ресурсов в сезонном разрезе, материальные балансы зерна (производство — расход на семена, корм, продовольствие, экспорт)

**Имитационные модели:**
- Воспроизводят поведение системы путём компьютерного эксперимента, позволяют анализировать сценарии «что если»
- Математический аппарат: алгоритмы, генераторы случайных чисел (метод Монте-Карло), системная динамика
- Примеры: имитация работы зернотока с учётом случайных поступлений зерна, моделирование развития стада КРС на 10 лет, моделирование влияния погоды на урожайность

### Вопрос 8. Коэффициенты прямых и полных материальных затрат. Использование в прогнозировании

**Коэффициент прямых затрат (aᵢⱼ):**

aᵢⱼ = xᵢⱼ / Xⱼ

Показывает, сколько единиц продукции отрасли i непосредственно затрачивается на производство одной единицы продукции отрасли j.

Пример: если для производства 1 т хлеба нужно 0,75 т зерна, то a(зерно, хлеб) = 0,75.

**Коэффициент полных затрат (bᵢⱼ):**

Матрица полных затрат B = (E − A)⁻¹

bᵢⱼ показывает, сколько всего (прямо и косвенно, через цепочки межотраслевых связей) нужно произвести продукции отрасли i для выпуска единицы конечного продукта отрасли j.

Пример: для 1 т хлеба нужно 0,75 т зерна напрямую, но для производства зерна тоже нужны ресурсы других отраслей, и т.д. Полные затраты зерна на хлеб будут больше 0,75.

**Использование в прогнозировании:**
- Если задан вектор конечного спроса Y (сколько продукции каждой отрасли нужно конечным потребителям), можно рассчитать необходимый валовой выпуск: **X = B · Y = (E − A)⁻¹ · Y**
- Позволяет оценить мультипликативный эффект: как рост спроса на продукцию одной отрасли повлияет на все остальные
- Используется для прогноза потребности в ресурсах при заданных плановых объёмах конечного потребления

### Вопрос 9. Математическая модель МОБ в матричной форме. Экономический смысл. Условия существования

**Балансовое уравнение для каждой отрасли:**

Xᵢ = Σⱼ xᵢⱼ + Yᵢ (i = 1, ..., n)

Валовой выпуск = межотраслевое потребление + конечный продукт

**Через коэффициенты прямых затрат:**

xᵢⱼ = aᵢⱼ · Xⱼ, поэтому:

Xᵢ = Σⱼ aᵢⱼ · Xⱼ + Yᵢ

**В матричной форме:**

**X = A·X + Y**

или

**(E − A)·X = Y**

откуда

**X = (E − A)⁻¹ · Y**

**Экономический смысл элементов:**
- **X** (вектор n×1) — валовые выпуски отраслей
- **A** (матрица n×n) — матрица коэффициентов прямых затрат (технологическая матрица)
- **Y** (вектор n×1) — конечный продукт (потребление, инвестиции, экспорт)
- **E** — единичная матрица
- **(E − A)** — матрица Леонтьева
- **(E − A)⁻¹** — матрица полных затрат (матрица Леонтьева обратная)

**Условия существования решения (условия продуктивности модели Леонтьева):**

1. **Критерий Хокинса–Саймона:** все главные миноры матрицы (E − A) положительны
2. **Эквивалентно:** наибольшее собственное число матрицы A меньше 1 (λmax(A) < 1)
3. **Экономический смысл:** каждая отрасль производит больше, чем потребляет — экономика способна обеспечить положительный конечный продукт
4. **Достаточное условие:** сумма элементов каждого столбца матрицы A меньше 1 (Σᵢ aᵢⱼ < 1 для всех j)

### Вопрос 10. Динамическая модель МОБ. Возможности по сравнению со статической

**Статическая модель:** X = A·X + Y (один период, без учёта инвестиций).

**Динамическая модель (Леонтьева):**

**X(t) = A·X(t) + B·[X(t+1) − X(t)] + Y(t)**

где:
- X(t) — валовой выпуск в периоде t
- A — матрица прямых текущих затрат
- B — матрица капитальных коэффициентов (фондоёмкости), bᵢⱼ — прирост фондов отрасли j за счёт продукции отрасли i на единицу прироста выпуска отрасли j
- B·[X(t+1) − X(t)] — инвестиции, необходимые для расширения производства
- Y(t) — конечное потребление (без инвестиций) в периоде t

**Дополнительные возможности динамической модели:**

| Возможность | Пояснение |
|-------------|-----------|
| Учёт инвестиций | Определяет, сколько продукции нужно направить на расширение производства |
| Траектория развития | Показывает путь от текущего состояния к целевому по периодам |
| Сбалансированный рост | Можно найти темп роста, при котором пропорции экономики сохраняются |
| Учёт лагов | Инвестиции текущего периода дают отдачу в будущих периодах |
| Прогнозирование | Позволяет строить прогноз на несколько лет вперёд с учётом накопления |

### Вопрос 11. Возможности и ограничения МОБ для прогнозирования АПК

**Возможности:**
- Оценка межотраслевых связей в АПК (растениеводство → кормопроизводство → животноводство → переработка)
- Прогноз необходимого валового выпуска при заданном конечном спросе
- Выявление «узких мест» — отраслей, не обеспечивающих потребности
- Расчёт мультипликативных эффектов (как рост спроса на мясо влияет на зерно, комбикорма, энергетику)
- Обоснование инвестиционных программ (динамический МОБ)

**Ограничения:**
- Постоянство коэффициентов прямых затрат (в реальности они меняются из-за НТП, изменения технологий)
- Линейность модели — не учитывает эффект масштаба
- Сложность получения достоверных данных для построения таблицы МОБ в АПК
- Не учитывает сезонность — критичную для с/х
- Не учитывает неопределённость (погода, цены, урожайность)
- Высокая агрегированность — одна отрасль объединяет разнородные производства
- Не отражает рыночные механизмы (спрос и предложение, конкуренция)

---

### Вопрос 12. Расчёт цен на основе МОБ. Допущения

**Стоимостной баланс (по столбцам):**

Xⱼ = Σᵢ xᵢⱼ + Vⱼ + Mⱼ

Валовой выпуск = промежуточные затраты + оплата труда + прибавочный продукт (прибыль, налоги, рента)

**Модель ценообразования:**

pⱼ = Σᵢ aᵢⱼ · pᵢ + vⱼ + mⱼ

или в матричной форме:

**p = Aᵀ · p + (v + m)**

откуда:

**p = (E − Aᵀ)⁻¹ · (v + m)**

где p — вектор цен, v — удельная оплата труда, m — удельная прибыль.

**Допущения:**
- Единые цены для всех потребителей одного продукта
- Постоянство коэффициентов прямых затрат (не зависят от масштаба)
- Затраты на оплату труда и прибыль заданы экзогенно
- Отсутствие монопольного ценообразования
- Не учитываются транспортные расходы на доставку

**Применение:** обоснование закупочных цен на с/х продукцию, оценка влияния роста цен на энергоносители на себестоимость с/х продукции.

### Вопрос 13. Учёт импортно-экспортных операций в МОБ

**Модификация балансового уравнения:**

Xᵢ + Zᵢ = Σⱼ xᵢⱼ + Yᵢ + Eᵢ

где Zᵢ — импорт, Eᵢ — экспорт продукции отрасли i.

**Способы учёта:**

1. **Конкурентный импорт** (импорт включён в общий ресурс): отечественная и импортная продукция рассматриваются как взаимозаменяемые. Коэффициенты прямых затрат рассчитываются по суммарному (отечественному + импортному) потреблению.

2. **Неконкурентный импорт** (отдельная строка): импортные товары, не производимые в стране, выделяются в отдельные строки. Коэффициенты затрат рассчитываются только по отечественной продукции.

3. **Разделение матрицы A:** A = Aᵈ + Aᵐ, где Aᵈ — затраты отечественной продукции, Aᵐ — затраты импортной.

**Влияние на коэффициенты полных затрат:**
- При конкурентном импорте коэффициенты полных затрат завышены (часть цепочки обеспечивается импортом)
- При учёте только отечественной продукции полные затраты точнее отражают нагрузку на национальное производство
- Изменение доли импорта меняет матрицу A и, следовательно, коэффициенты полных затрат

### Вопрос 14. Проблемы практического применения МОБ в современных условиях

**Проблемы:**
1. **Информационная:** сбор данных для таблицы МОБ требует колоссального объёма статистической информации; в условиях рыночной экономики предприятия не обязаны раскрывать данные
2. **Устаревание данных:** подготовка МОБ занимает 3–5 лет, за это время структура экономики меняется
3. **Нестабильность коэффициентов:** в переходной и кризисной экономике технологические коэффициенты быстро меняются
4. **Агрегирование:** укрупнение отраслей скрывает важные различия (разные виды зерна, мяса)
5. **Теневая экономика:** неучтённые потоки искажают баланс
6. **Линейность:** не отражает нелинейные рыночные механизмы

**Пути решения:**
- Использование выборочных обследований и косвенных методов оценки коэффициентов (метод RAS)
- Построение региональных МОБ меньшей размерности
- Комбинирование МОБ с эконометрическими моделями
- Применение IT для ускорения сбора и обработки данных
- Метод RAS для обновления устаревших таблиц МОБ на основе маржинальных данных

### Вопрос 15. Структура и принципы системы моделей оптимального планирования АПК

**Структура системы:**

| Блок | Содержание |
|------|-----------|
| Блок растениеводства | Оптимизация структуры посевных площадей, севооборотов, применения удобрений |
| Блок животноводства | Оптимальный рацион кормления, структура стада, размещение ферм |
| Блок кормопроизводства | Связь между растениеводством и животноводством: баланс кормов |
| Блок механизации | Оптимальный состав МТП, распределение техники по работам и срокам |
| Блок трудовых ресурсов | Баланс труда в сезонном разрезе |

**Принципы построения:**
1. **Системность** — все блоки взаимосвязаны (корма определяют поголовье; поголовье создаёт спрос на корма)
2. **Иерархичность** — модели разного уровня (хозяйство → район → регион)
3. **Итеративное согласование** — решения блоков взаимно уточняются
4. **Вариантность** — расчёт нескольких сценариев
5. **Адекватность** — модель должна отражать реальные технологии и ресурсы

### Вопрос 16. Двойственные оценки в задачах оптимального планирования с/х производства

**Двойственная оценка (теневая цена, маржинальная стоимость)** — показывает, на сколько изменится значение целевой функции при увеличении запаса соответствующего ресурса на единицу.

**Экономический смысл:**
- Двойственная оценка земли = 500 руб./га → дополнительный гектар увеличит прибыль на 500 руб.
- Двойственная оценка = 0 → ресурс недефицитный (есть излишек), его увеличение не влияет на прибыль
- Чем выше оценка, тем ценнее ресурс для хозяйства

**Использование в практике планирования:**
1. **Ранжирование ресурсов по дефицитности** — направлять инвестиции в расшивку самых ценных ограничений
2. **Оценка целесообразности приобретения ресурсов** — если рыночная цена ресурса меньше двойственной оценки, его выгодно приобрести
3. **Обоснование арендной платы** за землю (по двойственной оценке земельного ограничения)
4. **Ценообразование** — двойственные оценки как основа для расчёта внутрихозяйственных цен
5. **Согласование планов** разных уровней иерархии (через передачу двойственных оценок)

### Вопрос 17. Транспортная задача в оптимизации размещения производства и переработки с/х продукции

**Постановка:**

Имеется m пунктов производства (с/х предприятия, элеваторы) с объёмами aᵢ и n пунктов потребления (перерабатывающие предприятия, рынки сбыта) с потребностями bⱼ.

**Математическая модель:**

Целевая функция:
F = ΣᵢΣⱼ cᵢⱼ · xᵢⱼ → min

где cᵢⱼ — стоимость перевозки единицы продукции из пункта i в пункт j, xᵢⱼ — объём перевозки.

Ограничения:
- Σⱼ xᵢⱼ = aᵢ (весь объём из пункта i вывозится) — для i = 1,...,m
- Σᵢ xᵢⱼ = bⱼ (потребность пункта j удовлетворена) — для j = 1,...,n
- xᵢⱼ ≥ 0

Условие баланса: Σᵢ aᵢ = Σⱼ bⱼ (закрытая задача; если нет — вводится фиктивный пункт).

**Метод решения:** метод потенциалов (модификация симплексного метода для транспортных задач).

**Особенности для АПК:**
- Сезонность перевозок (уборка → пиковая нагрузка)
- Скоропортящаяся продукция (молоко, овощи) — ограничения по времени доставки
- Разные виды транспорта (автомобильный, железнодорожный)
- Учёт потерь при перевозке

### Вопрос 18. Оптимальное сочетание отраслей с/х предприятия. Специфические ограничения

**Задача:** определить объёмы производства каждой отрасли (культуры, виды скота), чтобы максимизировать прибыль при ограниченных ресурсах.

**Переменные:** xⱼ — объём j-й отрасли (площадь культуры, поголовье скота).

**Целевая функция:** F = Σⱼ cⱼ · xⱼ → max (чистый доход)

**Специфические ограничения:**

| Тип ограничения | Пример |
|----------------|--------|
| Земельные | Σ xⱼ ≤ S (общая площадь); доля пашни, сенокосов, пастбищ |
| Севооборотные | Доля зерновых ≤ 60% пашни; подсолнечник — не чаще раз в 7 лет; предшественники |
| Трудовые (сезонные) | Потребность в труде по месяцам ≤ наличие рабочей силы в каждом месяце |
| Кормовые | Производство кормов ≥ потребности поголовья (по видам: грубые, сочные, концентрированные) |
| Зоотехнические | Структура стада (доля коров ≥ 40%), нормы кормления по питательности |
| Финансовые | Затраты ≤ бюджет |
| Рыночные | Объём продаж ≤ ёмкость рынка; обязательные поставки по госконтрактам |
| Агротехнические | Минимальные/максимальные площади культур |

### Вопрос 19. Учёт риска и неопределённости в моделях оптимального планирования АПК

**Источники неопределённости в АПК:** погода, колебания цен, болезни растений/животных, изменение госполитики.

**Методы учёта:**

1. **Стохастическое программирование** — параметры модели (урожайность, цены) задаются как случайные величины с известным распределением. Оптимизируется математическое ожидание целевой функции.

2. **Робастная оптимизация** — поиск решения, устойчивого при наихудших возможных значениях параметров.

3. **Анализ чувствительности** — определение, как изменится оптимальный план при вариации параметров. Выделение критических параметров.

4. **Сценарный подход** — решение задачи для нескольких сценариев (оптимистический, пессимистический, базовый).

5. **Игровой подход (теория игр)** — природа как «противник»: строится матрица выигрышей для каждой стратегии при каждом состоянии природы. Критерии: Вальда (maximin), Гурвица, Сэвиджа (minimax regret), Лапласа.

6. **Диверсификация** — модель с ограничением на минимальное число отраслей (снижение риска за счёт разнообразия).

7. **Страховые ограничения** — резерв кормов, минимальные запасы семян.

---

### Вопрос 20. Многоцелевые модели оптимизации в АПК. Методы решения многокритериальных задач

**Проблема:** в АПК одновременно важны прибыль, занятость, экология, продовольственная безопасность — цели часто конфликтуют.

**Методы решения:**

1. **Свёртка критериев** — F = Σ wₖ · fₖ(x) → max, где wₖ — весовые коэффициенты важности k-го критерия. Проблема: субъективность весов.

2. **Лексикографический метод** — критерии ранжируются по приоритету. Сначала оптимизируется первый, затем второй с фиксацией первого и т.д.

3. **Метод ε-ограничений** — один критерий оптимизируется, остальные переводятся в ограничения (не хуже заданного уровня).

4. **Метод идеальной точки** — минимизация расстояния от вектора критериев до «идеальной точки» (максимумы каждого критерия по отдельности).

5. **Метод уступок** — ЛПР последовательно определяет допустимые уступки по каждому критерию.

6. **Парето-оптимальность** — поиск множества решений, где невозможно улучшить один критерий без ухудшения другого. ЛПР выбирает из Парето-множества.

**Примеры в АПК:** максимизация прибыли + минимизация экологического ущерба + максимизация занятости сельского населения.

### Вопрос 21. Модели оптимального планирования на региональном уровне АПК

**Отличия от уровня предприятия:**

| Аспект | Уровень предприятия | Региональный уровень |
|--------|--------------------|--------------------|
| Переменные | Площади культур, поголовье | Объёмы производства по районам, размещение предприятий |
| Ресурсы | Земля, техника, труд хозяйства | Совокупные ресурсы региона, межрайонный обмен |
| Ограничения | Внутрихозяйственные | + межрайонные перевозки, экология, соцзадачи |

**Дополнительные ограничения на региональном уровне:**
- Межрайонный баланс продукции (производство + ввоз ≥ потребность)
- Транспортные ограничения (пропускная способность дорог, расстояния)

- Продовольственная безопасность (самообеспечение по базовым продуктам)
- Инфраструктурные (мощности переработки, хранения)

**Целевые функции:**
- Максимизация валовой продукции региона
- Минимизация суммарных транспортных затрат
- Максимизация чистого дохода с учётом межрайонного обмена
- Многокритериальные: экономика + экология + социальная сфера

## Раздел 4: ЭММ для расчёта оптимального плана размещения предприятий АПК

### Вопрос 22. Математическая постановка задачи оптимального размещения предприятий по переработке с/х продукции

**Переменные:**
- yⱼ ∈ {0, 1} — строить (1) или не строить (0) предприятие в пункте j
- xᵢⱼ — объём продукции, перевозимой от поставщика i к предприятию j

**Целевая функция (минимизация суммарных затрат):**

F = Σⱼ Fⱼ · yⱼ + Σᵢ Σⱼ cᵢⱼ · xᵢⱼ + Σⱼ dⱼ(Σᵢ xᵢⱼ) → min

где:
- Fⱼ — капитальные затраты на строительство предприятия в пункте j
- cᵢⱼ — транспортные расходы на перевозку единицы сырья от i к j
- dⱼ — удельные затраты на переработку в пункте j

**Ограничения:**
- Σⱼ xᵢⱼ ≤ aᵢ (объём сырья у поставщика i)
- Σᵢ xᵢⱼ ≤ Mⱼ · yⱼ (мощность предприятия j, если оно построено)
- Σⱼ yⱼ · Mⱼ ≥ D (суммарная мощность покрывает общую потребность в переработке)
- xᵢⱼ ≥ 0, yⱼ ∈ {0, 1}

**Критерии оптимальности:**
- Минимум приведённых затрат (капитальные + эксплуатационные + транспортные)
- Минимум транспортных расходов (при фиксированных мощностях)
- Максимум прибыли перерабатывающих предприятий
- Многокритериальный: затраты + занятость + экология

### Вопрос 23. Оптимальная мощность перерабатывающего предприятия. Факторы

**Методика:**
Суммарные удельные затраты на единицу продукции:

C(M) = (F(M)/M) + p(M) + t(M) → min

где:
- M — мощность предприятия
- F(M)/M — удельные капитальные затраты (убывают с ростом M — эффект масштаба)
- p(M) — удельные эксплуатационные затраты (слабо зависят от M)
- t(M) — удельные транспортные затраты на доставку сырья (растут с M, т.к. зона снабжения расширяется)

Оптимальная мощность находится в точке минимума суммарных удельных затрат — баланс между экономией масштаба и ростом транспортных расходов.

**Факторы, влияющие на оптимальный размер:**
- Плотность сырьевой зоны (объём сырья на единицу площади)
- Транспортная инфраструктура (качество дорог, расстояния)
- Сезонность поступления сырья (для молокозаводов, сахарных заводов)
- Технология переработки (минимально эффективный масштаб)
- Скоропортящесть сырья (молоко — радиус ≤ 50 км)
- Спрос на готовую продукцию

### Вопрос 24. Целочисленное программирование для размещения предприятий АПК

**Суть:** в задачах размещения переменные «строить/не строить» принимают значения 0 или 1 (булевы), что требует методов целочисленного программирования.

**Методы решения:**
1. **Метод ветвей и границ** — дерево решений, последовательная фиксация переменных с отсечением бесперспективных ветвей
2. **Метод Гомори (отсечений)** — последовательное добавление линейных ограничений, отсекающих дробные решения

**Пример задачи:**

Определить, в каких из 5 потенциальных пунктов строить молокозаводы, чтобы минимизировать суммарные затраты (капитальные + транспортные) при условии переработки всего молока от 10 хозяйств-поставщиков.

Переменные: yⱼ ∈ {0,1} (j=1,...,5) — решение о строительстве; xᵢⱼ ≥ 0 — объём поставки молока.

**Особенности для АПК:**
- Комбинаторная сложность быстро растёт (2ⁿ вариантов для n пунктов)
- Необходимость учёта минимальной мощности (если yⱼ = 1, то Σᵢ xᵢⱼ ≥ Mⱼ_min)
- Дискретность решений отражает реальность: предприятие строится или нет

### Вопрос 25. Влияние транспортного фактора на размещение предприятий АПК

**Транспортный фактор** — один из решающих при размещении, особенно для перерабатывающих предприятий АПК.

**Два вида транспортных затрат:**
1. На доставку сырья (от хозяйств к предприятию) — зависят от радиуса сырьевой зоны
2. На доставку готовой продукции (от предприятия к потребителям)

**Методы учёта:**

1. **Транспортная задача** — оптимизация перевозок при фиксированном размещении (метод потенциалов)

2. **Задача размещения с транспортными затратами** — одновременная оптимизация местоположения и транспортировки (целочисленное ЛП)

3. **Задача Вебера** — нахождение точки, минимизирующей суммарные взвешенные расстояния до поставщиков и потребителей

4. **Гравитационные модели** — учёт притяжения к источникам сырья и рынкам сбыта пропорционально их объёмам и обратно пропорционально расстоянию

**Специфика АПК:**
- Сырьё рассредоточено по территории (в отличие от добывающих отраслей)
- Скоропортящесть ограничивает радиус доставки
- Сезонность — пиковые нагрузки на транспорт в уборочный период
- Потери при транспортировке (молоко, овощи, фрукты)
- Состояние сельских дорог — реальные расстояния больше геометрических

### Вопрос 26. Оптимизация размещения хранилищ с/х продукции

**Задача:** определить количество, местоположение и вместимость хранилищ для минимизации потерь и транспортных расходов при сезонном поступлении и равномерных поставках.

**Особенности:**
- Сезонность: продукция поступает в короткий период (уборка), а отпускается равномерно в течение года
- Необходимая суммарная вместимость определяется пиковым накоплением
- Потери при хранении зависят от условий (температура, влажность, вентиляция)

**Модель:**
Минимизировать: капитальные затраты на строительство хранилищ + транспортные расходы на доставку от полей и к потребителям + потери при хранении

При ограничениях:
- Суммарная вместимость ≥ максимальный объём единовременного хранения
- Равномерность поставок потребителям
- Радиус доставки ≤ допустимый
- Условия хранения (температурный режим) обеспечены

**Методы:** целочисленное программирование, транспортная задача, динамическое программирование (для оптимизации графика заполнения и опорожнения).

### Вопрос 29. Система моделей оптимизации производственной программы с/х предприятия

**Блоки системы:**

### Вопрос 30. Оптимизация структуры посевных площадей с учётом севооборотов

**Переменные:** xⱼ — площадь j-й культуры (или j-го севооборота)

**Целевая функция:** F = Σⱼ cⱼ · xⱼ → max (чистый доход)

**Математическая постановка с учётом севооборотов:**

Вместо отдельных культур вводятся севообороты как переменные: xₖ — площадь k-го севооборота.

Каждый севооборот — фиксированная ротация культур (например: пшеница → подсолнечник → ячмень → пар).

**Ограничения:**

| Тип | Формула | Смысл |
|-----|---------|-------|
| Земельные | Σₖ xₖ ≤ S | Общая площадь пашни |
| Агроэкологические | xₖ ≤ Sₖ_max | Площадь k-го севооборота ≤ площади пригодных почв |
| По культурам | Σₖ αⱼₖ · xₖ ≥ Qⱼ | Производство j-й культуры ≥ потребности (αⱼₖ — доля j-й культуры в k-м севообороте) |
| Трудовые | Σₖ tₖₘ · xₖ ≤ Tₘ | Труд в m-м месяце ≤ наличие |
| Кормовые | Производство кормовых культур ≥ потребности животноводства |
| Товарные | Производство товарных культур ≥ обязательств по контрактам |

### Вопрос 31. Оптимальное использование материально-технических ресурсов в АПК

**Задача:** распределить ограниченные материально-технические ресурсы (техника, удобрения, ГСМ, семена) между отраслями и работами для максимизации результата.

**Специфические ограничения:**

1. **Сезонность** — техника нужна в определённые агротехнические сроки (сев, уборка), а не равномерно
2. **Взаимозаменяемость** — разные марки тракторов могут выполнять одну работу с разной производительностью
3. **Комплектность** — трактор без сеялки бесполезен; ограничения по комплектации агрегатов
4. **Сроки выполнения** — агротехнические сроки жёстко ограничены (опоздание с уборкой → потери)
5. **Хранение и доставка** — ГСМ, семена, удобрения должны быть на месте к нужному сроку

**Модель:** задача ЛП с переменными xᵢⱼₜ — объём работы i, выполняемой техникой j в период t.

### Вопрос 32. Оптимизация рационов кормления с/х животных

**Задача:** составить рацион минимальной стоимости, обеспечивающий все зоотехнические нормы.

**Переменные:** xⱼ — количество j-го корма в рационе (кг/сутки на голову)

**Целевая функция:** F = Σⱼ pⱼ · xⱼ → min (стоимость рациона)

**Ограничения:**

| Тип | Формула | Смысл |
|-----|---------|-------|
| Питательность | Σⱼ aᵢⱼ · xⱼ ≥ Nᵢ | Содержание i-го питательного вещества (корм. ед., перевариваемый протеин, каротин и др.) ≥ нормы |
| Максимум по кормам | xⱼ ≤ dⱼ | Не более допустимого количества j-го корма (физиологические ограничения) |
| Минимум по кормам | xⱼ ≥ eⱼ | Не менее обязательного количества (например, грубых кормов) |
| Сухое вещество | Σⱼ sⱼ · xⱼ ≤ V | Объём сухого вещества ≤ вместимость ЖКТ |
| Соотношения | Грубые : сочные : концентрированные в заданных пропорциях |

**Наиболее эффективные модели:**
- ЛП (классический подход, метод симплексный)
- Параметрическое ЛП — анализ при изменении цен на корма
- Стохастическое ЛП — учёт вариации питательности кормов

### Вопрос 33. Оптимизация механизированных работ в растениеводстве. Учёт сезонности

**Задача:** определить состав МТП и распределение техники по работам и срокам для выполнения всех работ в агротехнические сроки с минимальными затратами.

**Переменные:** xᵢⱼₜ — объём i-й работы, выполняемой j-м агрегатом в период t.

**Учёт сезонности:**
- Год делится на агротехнические периоды (весенняя обработка, сев, уход, уборка, осенняя обработка)
- Каждая работа должна быть выполнена в свой период (жёсткие временные рамки)
- Техника может использоваться на разных работах в разные периоды
- В напряжённые периоды (уборка) потребность в технике максимальна

**Ограничения:**
- Σⱼ wᵢⱼ · xᵢⱼₜ ≥ Vᵢ — объём i-й работы выполнен полностью
- Σᵢ xᵢⱼₜ / Wⱼₜ ≤ Nⱼ · Dₜ — фонд рабочего времени j-го типа техники в период t ≤ наличие
- Агротехнические сроки: работа i выполняется только в допустимые периоды

**Целевая функция:** min затрат на содержание МТП + эксплуатационные расходы.

### Вопрос 34. Динамическое программирование для оптимизации сроков с/х работ

**Принцип оптимальности Беллмана:** оптимальная стратегия обладает свойством: каково бы ни было начальное состояние и начальное решение, последующие решения должны составлять оптимальную стратегию относительно состояния, возникшего в результате первого решения.

**Применение в АПК:**

1. **Оптимальные сроки уборки:** каждый день задержки — потери (осыпание зерна); каждый день раннего начала — недозрелое зерно. ДП находит оптимальный момент начала и последовательность уборки полей.

2. **Последовательность технологических операций:** обработка почвы → внесение удобрений → сев → уход → уборка. Оптимизация по критерию min потерь или min затрат с учётом зависимости между сроками операций.

3. **Оптимальные сроки замены техники:** в каждом году решение — ремонтировать или заменить. ДП минимизирует суммарные затраты на горизонте.

4. **Распределение ресурсов по периодам:** выделение воды для полива по декадам, распределение удобрений по подкормкам.

**Рекуррентное соотношение:** fₖ(s) = min_{u} [g(s,u) + fₖ₊₁(T(s,u))], где s — состояние, u — решение, g — затраты на шаге, T — функция перехода.

### Вопрос 35. Оптимизация параметров животноводческих комплексов

**Оптимизируемые параметры:**
- Поголовье (размер комплекса)
- Структура стада (доля коров, молодняка разного возраста, быков)
- Система кормления (тип рациона, уровень концентратов)
- Система содержания (привязное, беспривязное, пастбищное)

**Модель оптимизации структуры стада:**

Переменные: xⱼ — поголовье j-й половозрастной группы

Целевая функция: F = Σⱼ (выручка_j − затраты_j) · xⱼ → max

Ограничения:
- Воспроизводство стада: выход телят от коров, выбраковка, ремонт
- Кормовые: Σⱼ kᵢⱼ · xⱼ ≤ Kᵢ (i-го вида корма ≤ наличие)
- Помещения: Σⱼ sⱼ · xⱼ ≤ S (скотоместа)
- Трудовые: Σⱼ tⱼ · xⱼ ≤ T
- Зоотехнические: доля коров ≥ 40%, выход молодняка, сроки откорма

**Динамическая модель развития стада:**
- Оптимизация на несколько лет с учётом рождений, выбраковки, откорма, продаж
- Позволяет спланировать траекторию от текущего стада к оптимальному

### Вопрос 38. Оценка эффективности инвестиционных проектов в АПК с ЭММ

**Критерии эффективности:**

| Критерий | Формула | Адекватность для АПК |
|----------|---------|---------------------|
| NPV (чистая приведённая стоимость) | NPV = Σ CFₜ/(1+r)ᵗ − I₀ | Основной критерий, учитывает все денежные потоки |
| IRR (внутренняя норма доходности) | NPV(IRR) = 0 | Показывает устойчивость к изменению ставки |
| PI (индекс рентабельности) | PI = Σ CFₜ/(1+r)ᵗ / I₀ | Удобен для сравнения проектов разного масштаба |
| PP (срок окупаемости) | Σ CFₜ ≥ I₀ | Важен при высокой неопределённости |

**Использование ЭММ:**
1. **Имитационное моделирование (метод Монте-Карло):** случайная генерация параметров (урожайность, цены, затраты) → распределение NPV → оценка риска (вероятность убытка)
2. **Анализ чувствительности:** как изменится NPV при вариации ключевых параметров
3. **Сценарный анализ:** оптимистический / базовый / пессимистический сценарии
4. **Оптимизация портфеля проектов:** при ограниченном бюджете выбрать набор проектов с max суммарным NPV (задача о рюкзаке)
5. **Реальные опционы:** учёт гибкости — возможность расширить, сократить, отложить проект

**Наиболее адекватный критерий для АПК:** NPV с учётом Монте-Карло (длительный период окупаемости, высокая неопределённость урожайности и цен).

### Вопрос 39. Нечёткие модели и методы принятия решений в условиях неопределённости в АПК

**Нечёткая логика (fuzzy logic)** — математический аппарат для работы с неточными, приближёнными понятиями («высокая урожайность», «засушливый год», «хорошее качество почвы»).

**Основные понятия:**
- **Нечёткое множество** — множество с функцией принадлежности μ(x) ∈ [0, 1] (вместо 0 или 1)
- **Лингвистическая переменная** — переменная, значения которой — слова («низкая», «средняя», «высокая»)
- **Нечёткие правила** — ЕСЛИ урожайность высокая И цена низкая, ТО прибыль средняя

**Применение в АПК:**

1. **Нечёткое ЛП:** коэффициенты и/или ограничения заданы нечётко (урожайность «около 30 ц/га»)
   - Решение: α-срезы, перевод в серию чётких задач

2. **Нечёткие экспертные системы:** рекомендации по внесению удобрений, защите растений на основе нечётких правил

3. **Оценка земель и почв:** качество почвы — нечёткий показатель (зависит от многих размытых факторов)

4. **Принятие решений при неполной информации:**
   - Нечёткая матрица решений (вместо точных оценок — интервалы или нечёткие числа)
   - Нечёткий критерий Гурвица, нечёткий TOPSIS

5. **Управление поливом:** нечёткие контроллеры (вход: влажность почвы, температура → выход: объём полива)

**Преимущества для АПК:** возможность работать с неточными, экспертными данными, которые типичны для сельского хозяйства.

### Вопрос 40. Адаптивные модели управления с/х производством

**Адаптивная модель** — модель, параметры которой корректируются по мере поступления новой информации (погода, рыночные цены, состояние посевов).

**Принципы:**
1. **Обратная связь** — фактические результаты сравниваются с плановыми, отклонения → корректировка
2. **Скользящее планирование** — план пересматривается на каждом шаге (неделя, месяц, сезон)
3. **Обучение** — модель учится на исторических данных и улучшает прогнозы

**Методы:**

| Метод | Применение |
|-------|-----------|
| Адаптивное прогнозирование (экспоненциальное сглаживание, адаптивные фильтры) | Прогноз урожайности, цен с автокоррекцией по мере поступления данных |
| Стохастическое динамическое программирование | Оптимальные решения на каждом этапе с учётом случайных факторов |
| Марковские модели принятия решений (MDP) | Управление стадом, запасами в условиях неопределённости |
| Reinforcement Learning | Адаптивное управление поливом, внесением удобрений |
| Модели с обратной связью | Двухэтапное стохастическое программирование: план → реализация погоды → корректировка |

**Учёт изменчивости:**
- Погодно-климатические условия: оперативная корректировка плана по метеопрогнозу
- Рыночная конъюнктура: пересмотр структуры реализации при изменении цен
- Биологические процессы: коррекция рационов по результатам контрольных доений, коррекция защиты растений по фитомониторингу
