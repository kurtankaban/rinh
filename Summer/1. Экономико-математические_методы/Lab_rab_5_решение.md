# Лабораторная работа №5

## Тема 2.1: Классификация экономико-математических моделей и этапы их построения

### Задание 1

**Условие:** Фермер имеет 100 га пашни и хочет посеять пшеницу и подсолнечник.

| Показатель | Пшеница | Подсолнечник |
|------------|---------|--------------|
| Прибыль, руб./га | 6 000 | 10 000 |
| Трудозатраты, чел.-ч/га | 20 | 40 |

Трудовые ресурсы ограничены: не более 3 000 чел.-ч.

**Цель:** распределить землю между культурами так, чтобы получить максимальную прибыль.

---

### Решение

#### 1. Определяем переменные

- x₁ — площадь под пшеницу (га)
- x₂ — площадь под подсолнечник (га)

#### 2. Целевая функция (максимизация прибыли)

**F = 6000·x₁ + 10000·x₂ → max**

#### 3. Система ограничений

| № | Ограничение | Смысл |
|---|-------------|-------|
| 1 | x₁ + x₂ ≤ 100 | Площадь пашни не более 100 га |
| 2 | 20·x₁ + 40·x₂ ≤ 3000 | Трудовые ресурсы не более 3 000 чел.-ч |
| 3 | x₁ ≥ 0, x₂ ≥ 0 | Условия неотрицательности |

#### 4. Классификация модели

| Признак | Тип |
|---------|-----|
| По цели | Оптимизационная (максимизация прибыли) |
| По учету времени | Статическая (один период) |
| По мат. аппарату | Линейное программирование |
| По характеру данных | Детерминированная (все данные известны точно) |

#### 5. Решение задачи

Найдём угловые точки области допустимых решений.

**Граничные прямые:**
- (1): x₁ + x₂ = 100
- (2): 20x₁ + 40x₂ = 3000, т.е. x₁ + 2x₂ = 150

**Точки пересечения:**

**Точка A (x₂ = 0 в обоих):**
- Из (1): x₁ = 100, x₂ = 0
- Проверка (2): 20·100 = 2000 ≤ 3000 — выполняется
- A = (100, 0)

**Точка B (пересечение (1) и (2)):**
- x₁ + x₂ = 100
- x₁ + 2x₂ = 150
- Вычитаем: x₂ = 50, x₁ = 50
- B = (50, 50)

**Точка C (x₁ = 0 в (2)):**
- 2x₂ = 150, x₂ = 75
- Проверка (1): 0 + 75 ≤ 100 — выполняется
- C = (0, 75)

**Точка O = (0, 0)**

**Значения целевой функции в угловых точках:**

| Точка | x₁ | x₂ | F = 6000x₁ + 10000x₂ |
|-------|----|----|----------------------|
| O | 0 | 0 | 0 |
| A | 100 | 0 | 600 000 |
| **B** | **50** | **50** | **800 000** |
| C | 0 | 75 | 750 000 |

#### 6. Оптимальный план

**x₁* = 50 га (пшеница), x₂* = 50 га (подсолнечник)**

**Fmax = 800 000 руб.**

#### 7. Анализ использования ресурсов

| Ресурс | Доступно | Использовано | Остаток |
|--------|----------|-------------|---------|
| Пашня | 100 га | 50 + 50 = 100 | 0 (использован полностью) |
| Труд | 3 000 чел.-ч | 20·50 + 40·50 = 3 000 | 0 (использован полностью) |

Оба ресурса использованы полностью — они являются дефицитными.

#### 8. Этапы построения модели (в данной задаче)

1. **Постановка задачи** — определили цель: максимизация прибыли фермера
2. **Определение ограничений** — пашня (100 га), труд (3000 чел.-ч)
3. **Введение переменных** — x₁ (пшеница), x₂ (подсолнечник)
4. **Целевая функция** — F = 6000x₁ + 10000x₂ → max
5. **Ограничения** — записаны в виде линейных неравенств
6. **Решение** — перебор угловых точек допустимой области

## Тема 2.2: Балансовые экономико-математические модели

### Задание 2

**Условие:** В регионе действуют две отрасли:
- Отрасль 1 — сельское хозяйство (зерно)
- Отрасль 2 — пищевая промышленность (хлебобулочные изделия)

**Данные:**
- Сельское хозяйство произвело 120 т зерна
- Пищевая промышленность произвела 80 т хлебобулочных изделий
- Из зерна: 20 т — на собственные нужды с/х, 60 т — пищевой промышленности
- Из хлебобулочных изделий: 10 т — сельскому хозяйству, 5 т — на собственные нужды

---

### Решение

#### 1. Межотраслевой баланс (таблица)

Обозначения:
- xᵢⱼ — объём продукции отрасли i, использованной в отрасли j
- Yᵢ — конечный продукт отрасли i
- Xᵢ — валовой выпуск отрасли i

| Отрасль-производитель | Потребление в отрасли 1 (с/х) | Потребление в отрасли 2 (пищ.) | Конечный продукт (Yᵢ) | Валовой выпуск (Xᵢ) |
|----------------------|-------------------------------|-------------------------------|----------------------|---------------------|
| 1 (с/х, зерно) | x₁₁ = 20 | x₁₂ = 60 | Y₁ = 40 | X₁ = 120 |
| 2 (пищ., хлеб) | x₂₁ = 10 | x₂₂ = 5 | Y₂ = 65 | X₂ = 80 |

#### 2. Проверка баланса

Для каждой отрасли: **Xᵢ = xᵢ₁ + xᵢ₂ + Yᵢ**

- Отрасль 1: 20 + 60 + Y₁ = 120 → **Y₁ = 40 т**
- Отрасль 2: 10 + 5 + Y₂ = 80 → **Y₂ = 65 т**

#### 3. Коэффициенты прямых затрат (матрица A)

aᵢⱼ = xᵢⱼ / Xⱼ — сколько единиц продукции отрасли i нужно для производства единицы продукции отрасли j.

| | Отрасль 1 | Отрасль 2 |
|---|-----------|-----------|
| Отрасль 1 | a₁₁ = 20/120 ≈ 0,167 | a₁₂ = 60/80 = 0,750 |
| Отрасль 2 | a₂₁ = 10/120 ≈ 0,083 | a₂₂ = 5/80 = 0,0625 |

Матрица A:
```
A = | 0,167  0,750 |
    | 0,083  0,063 |
```

#### 4. Матричная форма модели Леонтьева

**X = A·X + Y**, или **(E − A)·X = Y**

где E — единичная матрица.

```
(E - A) = | 0,833  -0,750 |
           | -0,083  0,938 |
```

#### 5. Конечный продукт

| Отрасль | Валовой выпуск | Межотраслевое потребление | Конечный продукт | Доля конечного продукта |
|---------|---------------|--------------------------|------------------|------------------------|
| 1 (с/х) | 120 т | 20 + 60 = 80 т | **40 т** | 40/120 = **33,3%** |
| 2 (пищ.) | 80 т | 10 + 5 = 15 т | **65 т** | 65/80 = **81,25%** |

#### 6. Вывод

**Пищевая промышленность (отрасль 2) значительно больше ориентирована на конечное потребление** — 81,25% её продукции идёт конечным потребителям. Сельское хозяйство отдаёт на конечное потребление лишь 33,3%, а основная часть зерна (66,7%) используется как промежуточный продукт (семена, корм, сырьё для пищевой промышленности).

Это типичная структура: сельское хозяйство выступает поставщиком сырья для перерабатывающей промышленности, а пищевая промышленность — конечным звеном цепочки, производящим продукцию для населения.
