## Курсовая работа

Написать **курсовой проект**.

Тема — на выбор из файла `Темы курсовых проектов.md`.

Для оформления можно использовать шаблон `Рамка.drawio`.

---

## Для опоздавших с лабораторными

Студентам, которые **не сдадут лабораторные работы вовремя**, на зачёте/экзамене потребуется **отвечать на вопросы** из файла:

- `Вопросы к экзмену.md`

Рекомендуется заранее подготовить ответы по всем вопросам из списка.
